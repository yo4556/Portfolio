SAE 203

Projet web développé en HTML, CSS et PHP, intégrant une base de données MariaDB pour la gestion dynamique des contenus.

Détails du Projet

Cette SAÉ avait pour objectif la réalisation d'un site web à thème libre répondant à une problématique précise dans un contexte professionnel. J'ai utilisé les technologies suivantes : HTML, CSS, PHP ainsi qu'une base de données MariaDB.

Réalisé dans le cadre de ma formation en Métiers du Multimédia et de l'Internet, ce projet m'a permis de développer mes compétences en développement front-end, mais surtout en logique avec PHP, ainsi qu'en gestion de base de données.

Dans le cadre de ce projet, une partie du site devait être sécurisée, notamment l'espace administrateur. Nous avons ainsi mis en place un système de protection contre les attaques courantes, notamment sur le formulaire d'authentification, afin de garantir la sécurité des accès à l'interface admin.

Cette interface permettait entre autres de télécharger des images, stockées sur le serveur, dont les chemins étaient enregistrés dans la base de données MariaDB. Elle donnait également la possibilité de gérer dynamiquement les contenus : ajout, modification ou suppression des données à tout moment.

Fonctionnalités clés

  • Responsive Design (compatible mobile, tablette, desktop)
  • Navigation intuitive
  • Intégration de contenus multimédias (images, vidéos, etc.)
  • Espace administrateur sécurisé
  • Gestion dynamique des contenus via base de données

Technologies utilisées

HTML5 CSS3 PHP MariaDB / phpMyAdmin
